Setup Git
$ git config --global user.name "<name>"
$ git config --global user.email <email>

Initialize git repo
$ git init

Project Status
$ git status

Add file to stage
$ git add <filename>

Commit file
$ git commit

	options 
	-m "message" //add message to commit
	-a 	     //add files to stage

Check repo history
$ git log
	options
	--graph		//branch graph
	--oneline	
	-p		//changes
	-1 		//number of commits
	--decorate

Create alias
$ git config --global alias.lg 'log --graph -- oneline --decorate -- all'
